# RetailTrack

A Flutter-based order taking and retail management application with Google Sheets integration and offline-first architecture.

## üìã Overview

RetailTrack is a cross-platform Flutter application designed to streamline retail order management. It enables users to:
- Create and manage orders for clients
- Manage product inventory
- Maintain client information
- Sync data bidirectionally with Google Sheets
- Work offline with automatic synchronization when connectivity is restored

## ‚ú® Features

### Core Functionality
- **Order Management**: Create, view, and track orders with detailed information
- **Product Catalog**: Manage products with import/export capabilities
- **Client Management**: Maintain customer information and relationship tracking
- **Google Sheets Integration**: Automatic bidirectional sync with Google Sheets
- **Offline Support**: Full offline capability with automatic sync when online
- **Data Import**: Import products and clients from Excel files
- **Material Design 3**: Modern, responsive UI using Material Design 3

### Key Capabilities
- **Authentication**: Google Sign-in integration for secure access
- **Database**: Local SQLite database for offline-first architecture
- **Connectivity Awareness**: Automatic detection and handling of network changes
- **Data Synchronization**: Smart bi-directional sync preventing data loss
- **Cross-Platform**: Supports Android, iOS, Web, and macOS
